Create Issue From Task Odoo App

This Odoo app helps user to create issue from project task, User can easily create and access issue from project task.

Features

Create Issue from Task.

User can easily create issue from project task..

Easy to Access Issue.

User can easily access project task issue from "Issue" smart button.
Demo Request Video Tutorial

Create Issue from Task

Under task "Action" menu user can see menu to create issue from task.

Create Issue Wizard

On clicking menu new wizard will open, Here user have to enter name for issue.

Access Issue from Smart Button

User can see smart button on task with created issue from task.

Issue Tree View

User can see all created issue from tak in tree view.

Project Issue

User can see create issue from project task in from view.
