===================
Project Agile Scrum
===================

This module is plugin for Project Agile, which enables user to manage
projects with Scrum methodology. It extends team by introducing concepts
of Scrum Master, Sprints, Sprint velocity.

Models introduced:
- Project Agile Scrum Sprint
