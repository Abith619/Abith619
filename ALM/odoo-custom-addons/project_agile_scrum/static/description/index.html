Project Agile Scrum

Overview

This module is plugin for Project Agile, which enables user to manage projects with Scrum methodology. It extends team by introducing concepts of Scrum Master, Sprints, Sprint velocity.

Although modules around Project Agile can work using standard Odoo Web UI, it is recommended to use Scrummer UI which is designed specially for Agile methodologies.
If you are interested in Scrum methodology, take a look at Scrummer Scrum module.

Models introduced:

  • Project Agile Scrum Sprint
