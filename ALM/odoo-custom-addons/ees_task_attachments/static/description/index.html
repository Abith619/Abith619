Add attachments to your tasks

Upload and control files in each task of your Odoo projects

This module adds the following features to the "Project" app:

  • Attachment uploading in project tasks and sub-tasks
  • Direct access to task attachments through a new button in the task's form view
  • Labels attachments according to which task they belong to, useful in the project's attachments view

Individual Task Attachments View

Each task has its own attachments view

Full Sub-Task Support

Supports attaching files to any sub-task
