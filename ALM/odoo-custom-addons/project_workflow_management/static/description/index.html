Project Workflow

Overview

This module provides functionality to create fully configurable workflow around Project Task

Usage

To manage project workflows got to: Project -> Configuration -> Workflow -> Workflow

Workflows are composed of workflow states

Workflow states are wrappers around Tasks stage.
Incoming and outgoing transitions defines routes that task may have during lifecycle. Is global field can be used when Task should be able to transit to or from that state from any other state.

Workflows can be manipulated from diagram view.
States of type ToDo are blue, In progress are yellow, and Done are green
