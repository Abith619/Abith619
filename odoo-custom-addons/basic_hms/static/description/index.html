Basic Hospital Management System Odoo Apps

This Odoo Apps helps to manage day to day activities of hospital and clinic which can be helpful for any Hospital and medical facilities. This module helps to manage Patient details,Doctors details, Patient Appointment, Prescription, Patient Hospitalization etc

Features

Manage Patient Details.

The Patient can enter their details.There are tabs will shows patient details like medicament, diseases, lab test, etc.

Receptionist can create Appointment.

Receptionist can create appointment with the details like physician , consultation service etc.

Patient Hospitalization Details.

Help administrative department to save the details of patient about his/her hospitalization like bed details, hospitalization date & expected discharge date, and other.

Generate Patient's Prescriptions.

Generate prescriptions with the doctor's and patient's details, can create invoice from prescription.

Lab Test Request and Result.

Patient can request for the lab test and can get the user of the lab test..

Generate Report for patient details, appointment, prescriptions, lab-test.

Patient can generate the report for their details, appointment, prescriptions, lab-test.

Appointments

From this tab user can create an appointment for a patient with basic details like physician , consultation service etc..


Here we have feature showing calendar to show appointments on particular dates.


Prescriptions

Prescription is feature where after diagnosis of the patient , you can enter the details of prescripted medicaments, its dosage and other details.


Prescription Report

Reports for Prescription can be generated.


Patient Hospitalization

This form will help administrative department to save the details of patient about his/her hospitalization. Patients bed details, hospitalization date & expected discharge date, and other details can be saved here.


Once "confirmed registration" registration will be in confirmed stage. From this stage it can be cancelled.


If patient gets admitted to hospital then you will see discharge patient button.
